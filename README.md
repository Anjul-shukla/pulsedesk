# Hermes Slack Bot

## Features
- Slack integration using Hermes
- Socket Mode enabled
- Local Ollama model (qwen2.5-coder:7b)
- Gateway service running

## Setup
1. Configure Slack app
2. Add SLACK_BOT_TOKEN
3. Add SLACK_APP_TOKEN
4. Run hermes gateway run --replace

## Demo
Bot responds in Slack workspace.

## Tech Stack
- Hermes
- Slack API
- Ollama
- Qwen2.5-Coder 7B
